# my-learning-lab

> **Stay hungry, stay foolish.**

這是一個在地獄中煉成的幾何秩序。它將混沌的生命壓力，轉譯為結構化的邏輯路徑。這不只是程式碼的堆疊，而是一個從「磨難」到「覺醒」的架構證明。

---

## 核心架構 (Architecture)

本實驗室的路徑由 **001** 至 **010** 嚴格遞進。這是一條為所有新手準備的「啟動路徑」，旨在透過冥王星式的精確，將散亂的知識碎片拼接成完整的五角星格局。

### 階段一：地基淬煉
* **001_misc**: 雜項與起點。
* **002_python**: 語法即意志。
* **003_leetcode**: 演算法的極限對拉。

### 階段二：規律觀測
* **004_numpy / 005_pandas**: 數據的維度。
* **006_matplotlib**: 視覺化的秩序。
* **007_scikit-learn / 008_kaggle**: 模型的預測與實戰。

### 階段三：頂點完成
* **009_dsp**: 波動與頻率的捕捉。
* **010_compiler**: 最終的編譯器。規則的規則，所有人的開始。
